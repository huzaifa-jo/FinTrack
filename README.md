# FinTrack 

A personal finance tracker built using HTML, CSS, and JavaScript.

## Features
- Track income and expenses
- Manage debts (I owe / Owed to me)
- Dashboard with disposable income
- Recent transactions table
- LocalStorage data persistence

## Tech Used
- HTML
- CSS
- JavaScript (Vanilla)
